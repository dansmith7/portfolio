# Логотипы компаний

Переименуйте файлы логотипов в папке `/public/logos/` следующим образом:

1. `rostelecom.png` - Ростелеком (фиолетово-оранжевый логотип)
2. `dit-moscow.png` - Департамент информационных технологий города Москвы (коричневый логотип)
3. `sber.png` - Сбер Банк (зеленый логотип с галочкой)
4. `rzd.png` - РЖД (красный логотип с буквами РЖД)
5. `bank-of-russia.png` - Банк России (серый логотип с двуглавым орлом)

**Формат:** PNG с прозрачным фоном
**Размер:** Логотипы будут автоматически масштабироваться до высоты 80px

Логотипы будут отображаться в белом цвете на темном фоне благодаря CSS фильтру.
